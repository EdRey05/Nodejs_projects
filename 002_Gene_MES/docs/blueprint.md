# **App Name**: LabFlow

## Core Features:

- Modular Step Execution: Run individual processing steps or combine them into automated pipelines, managed via a central workflow system.
- Excel Parsing & Export: Import experimental data from Excel files and export processed data back into Excel format. Utilize comprehensive parsing tools.
- AI-Powered Chatbot: Integrate an AI chatbot with contextual awareness of the current application state. Offers help, answers questions, and guides the user. It will act as a tool that will improve the usability of the tool.
- Interactive Plate Visualization: Display interactive 48 and 96-well plates with drag-and-drop functionality for samples.
- Sample Management: Drag and drop samples onto plate wells. Programmatically access sample information based on well location.
- Well Parameterization: Define and apply parameters to wells, such as antibiotics and temperature, visually represented through coloring.
- Custom Workflow Definition: Visually construct pipelines by selecting and sequencing modular steps from the left sidebar, including defining data flow between them.

## Style Guidelines:

- Primary color: Indigo (#4B0082) for a focused and scientific feel.
- Background color: Very light lavender (#F0E6FF) for a soft, clean backdrop.
- Accent color: Teal (#008080) to highlight key interactive elements.
- Body and headline font: 'Inter', a sans-serif font for a modern and neutral look.
- Use clear, concise icons to represent different experimental parameters (temperature, antibiotics, etc.).
- Left sidebar for module selection and a main content area for plate visualization and manipulation.
- Use subtle animations when dragging and dropping samples or applying parameters to wells.
